# FlaskPhishing
一个python编写的网络钓鱼工具，操作简单，界面简洁。 
10+国内外网站，持续更新,未来会补充更多。 


## 使用方法

### 1,下载py文件,链接:
```
https://github.com/sswz666/FlaskPhishing/archive/refs/heads/main.zip
```

### 2,安装第三方库flask:
```
pip install flask
```

### 3,运行FlaskPhishing.py:
```
python FlaskPhishing.py
```

### 4,输入信息:
- 输入网站文件夹:(例:github)
- 输入跳转真链接:(例:https://github.com)
